# Project

## コンセプト

初めての学生プロジェクトで、進め方の全体像が見通せるようになるWebメディア。

## 解決したい問題

学生プロジェクトの開始直後に、何を決め、何を話し、次に何をすればよいか分からない。

## 目的

プロジェクト経験の少ない学生が、開始直後に必要な話し合いや行動を理解し、自分たちで活動を進められるようにする。

## ゴール

初回から3回目程度の会議を通して、次の内容が暫定的に決まり、メンバー間で共有されている状態を目指す。

- プロジェクトの目的
- 役割
- 直近のタスク
- 次回の議題

## ターゲット

初めてプロジェクト活動に参加することになった高校生・大学生。

この企画の発端が制作者の中学・高校時代の経験であるため、高校生寄りに設計する。

## 対応する範囲

- プロジェクト開始直後
- 初回から3回目程度の会議
- 会議前の準備
- 目的、役割、直近タスク、次回議題の整理
- プロジェクト全体の流れを理解するための案内

## 対応しない範囲

現時点では、次の内容を初期研究・初期実装の中心には含めない。

- プロジェクト参加者の募集
- プロジェクト中盤以降の高度な進捗管理
- ログイン、アカウント、マイページ
- 複数人によるリアルタイム共同編集
- 学校や教員向けの管理画面
- 収益化機能
- コミュニティ機能

## 未確定事項

- ペルソナ
- 初回から3回目までの具体的な会議内容
- MVPに含める機能
- 各コンテンツの内容
- サイトマップ
- 各ページのデザイン
- 調査方法と対象者
