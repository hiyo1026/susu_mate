# すすメイト / susumate

> 学生プロジェクトの「はじめの迷い」を、旅の地図に変える学習メディア。

## 概要

「すすメイト」は、学生が主体となって進めるプロジェクトの走り始めを支援するWebサイトです。

企画立案、会議、役割分担、スケジュール、メンバーとのコミュニケーションなど、  
プロジェクトを始めたばかりの学生が迷いやすい内容を扱います。

すすメイトは、学生の代わりにプロジェクトを管理するサービスではありません。

- 全体の流れを知る
- 今いる段階を確認する
- 次にすることを見つける
- 記事で基本を学ぶ
- 支援ツールを実際に使う
- 自分たちのプロジェクトへ持ち帰る

という体験を通して、学生が自分たちで進むためのナビゲーション役を目指します。

---

## このリポジトリについて

このリポジトリには、すでにトップページの元となるファイルがあります。

```text
index.html
style.css
script.js
```

今後はこれらを土台として編集し、複数ページのWebサイトへ拡張します。

### 方針

- 既存トップページのデザインを活かす
- HTML・CSS・JavaScriptで制作する
- 1ページごとにHTMLファイルを追加する
- `style.css` と `script.js` を共通で使う
- ページ間を相対リンクで接続する
- 登録やログインは作らない
- GitHub Pagesなどの静的ホスティングで公開できる形にする

Astroなどのフレームワークへの移行は、現段階では行いません。

## 背景

オンライン環境が整い、学生が授業、文化祭、探究学習、部活動、コンテスト、創作活動などでプロジェクトへ参加する機会が増えています。

一方で、次のような悩みが起こりやすくなります。

- 会議で何を議題にすればよいか分からない
- 仲を深める方法が分からない
- 最初に話し出すのが緊張する
- 会議内の役割分担が分からない
- ネタ探しに苦戦する
- 計画がスムーズに進まない
- 専門用語が分からない
- 人と話すことが苦手

既存のプロジェクト管理サービスには、次のような難しさがあります。

- 社会人向けのものが多い
- 学生が気軽に利用しにくい
- 機能が多く迷いやすい
- ツールの使い分けが難しい
- そもそも何が必要か分からない

## ゴール

### 利用前

> プロジェクトをやることになったけれど、何から手をつければよいか分からない。

### 利用後

> プロジェクト全体の流れと次にすることが分かり、自分たちでも進められそうだと思える。

### 目的

基本的な進め方を押さえ、迷いを減らしながら会議とプロジェクトを進められるようになること。

## デザインコンセプト

### テーマ：旅路 × 空・宇宙

プロジェクトを、まだ知らない場所へ進む旅として表現します。

- 旅路：進行、現在地、目的地、次の一歩
- ロードマップ：全体像、段階、チェックポイント
- 空・宇宙：未知への挑戦、可能性、広がり
- 仲間：答えを押しつけず、隣で案内する存在
- 夜明け：迷いが減り、見通しを持てた状態

### ユーザーに抱いてほしい感情

1. 見通しが持てる
2. 迷いが減る
3. 自分たちでも進められそう
4. 未知のことへ挑戦してみたい
5. 少しワクワクする

### 現在のデザイン

既存の `style.css` では、夜の宇宙から夜明けへ進む雰囲気が表現されています。

主な特徴：

- 深いネイビーの背景
- オレンジ色の夜明け
- ミント色のオーロラ
- 星や軌道の装飾
- 明朝体とゴシック体の組み合わせ
- ロードマップが光っていくスクロール演出
- モバイルナビゲーション
- 動きを減らす設定への対応

新しいページも、この世界観を基本にします。

## 開発方法

ビルド作業はありません。

HTMLを直接開くこともできますが、ページ遷移やJavaScriptを正しく確認するため、簡易サーバーの使用を推奨します。

### 起動

リポジトリのフォルダで実行：

```bash
python3 -m http.server 8000
```

ブラウザで開く：

```text
http://localhost:8000/
```

### 終了

ターミナルで：

```text
Control + C
```
